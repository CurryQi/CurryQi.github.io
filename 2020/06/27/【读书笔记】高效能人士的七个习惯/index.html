【读书笔记】高效能人士的七个习惯.md
发表于 2020-06-27| 更新于 2020-06-27|读书笔记
阅读量:|评论数:

习惯一：积极主动

  1. 行为主动，为自己过去、现在及未来的行为负责，依据原则及价值观，而非情绪或外在环境来下决定。
  2. 积极主动的人是改变的催生者，摒弃被动的受害者角色，不怨天尤人。
  3. 发挥了人类思想独特的禀赋：自我意识、良知、想象力和独立意识，由内而外的方式创造改变，积极面对一切。
  4. 积极主动选择创造自己的人生，这也是每个人最基本的决定。

思考：

积极主动是一种向上的心态和价值观。

  • 就像是学生时代的，由要我学，转变为我要学一样；
  • 对于刚步入社会的我们而言，更是需要有这样一种积极主动的态度；
  • 要我去完成，变成我要去完成；要求我做得好，变成我要求做得好。

拥有了这样的态度之后，也将会拥有对工作的热情，把工作当成实现自我价值的台阶，而不仅仅是一项任务、混日子的手段。

习惯二：以终为始

  1. 所有的计划都应该经过两次创造：显示在大脑中酝酿，其次才是实质的创造。
  2. 个人、家庭、团队和组织在做任何计划时，均先拟出愿景和目标，并以此塑造未来，全心投入自己最重视的原则中、价值观、关系及目标。
  3. 对个人、家庭或组织而言，使命宣言可以说是愿景的最高形式，它是根本的决策，主宰了所有其他决定。
  4. 领导工作的核心，就是基于共有的使命、愿景和价值观，创造出一个文化。

思考：

以终为始是一种闭环的思考模式。

  • 对于步入职场的我，五年后，我将何去何从，究竟是继续在目前的岗位上，兢兢业业，写着脚本，编着用例，获取当一个资格老师傅？
  • 还是转向偏管理，学习一些管理相关的内容？
  • 还是作为花这五年时间，努力钻研，成为技术专家？

对于跨专业、跨技能入职的我来说，称为技术专家或许挑战更大一些。

那就在这儿立一个Flag吧，以此为证，看看五年后我能否成为我目前所在领域下某一方向的专家。

既然要以终为始，目前的终点为成为某一技术方向的专家，那就不能仅仅是浮于表面，满足工作上的代码、任务，还需要去补充更多的通信知识、原理、细节。了解开发底层对于这一块儿相关的实现方式，等等，抓紧时间。

习惯三：要事第一

  1. 实质的创造，是梦想（你的目标、愿景、价值观及要事处理顺序）的组织和实践。
  2. 次要的事不必摆在第一位，重要的事也不能放在第二位。
  3. 无论迫切性如何，个人和组织均要更多的聚焦要事，重要的是，把要事放在第一位。

思考：

要事第一是指重要的事，而不是紧急的事。

  • 要能够明白什么是要事，当前做的事，根本目的是为了什么。
    1. 学习的目的是为了更好的工作和实现自我价值
    2. 工作的目的是为了挣钱
    3. 玩游戏的目的是为了放松身心
  • 学习的态度要从广度出发，除了专业知识和技能外，还要学习其他人对人生的观察和看法，对世界的理解、对生活的理解，更重要的是对自身的理解。还有其他的例如道德伦理、为人处世等等。
  • 学习的行动要向深度进行，特别是在某个领域，应该有相应的深刻理解。

习惯四：双赢思维

  1. 双赢思维是一种基于互相尊敬、寻求互惠的思考框架和心意，目的是分享更多的机会、财富及资源，而非敌对式竞争。
  2. 双赢既非损人利己（赢输），亦非损己利人（输赢）。
  3. 我们的工作伙伴及家庭成员要从互赖式的角度来思考（“我们”，而非“我”）。
  4. 双赢思维鼓励我们解决问题，而不是推卸责任，协助个人找到互惠的解决办法，是一种咨询、力量、认可及报酬的分享。

习惯五：知彼解己

  1. 当我们不再急切回答，而是诚心去了解、聆听他人，才能够开启真正的沟通，增加彼此关系。
  2. 对方获得理解后，会觉得收到尊重和认可，进而卸下心里防备，坦然交谈，双方对彼此的了解也就更顺畅自然。
  3. 知彼需要仁慈心，解己需要勇气，平衡两者，可大幅提升沟通效率。

习惯六：统合综效

  1. 统合综效谈的是创造的第三种选择，即非按照我的方式，亦非遵循你的方式，而是创造第三种更好的方法。
  2. 它是互相尊重的成果——不但了解了彼此，甚至还称赞彼此的差异，欣赏对方解决问题及把握机会的手法。
  3. 个人的力量是团队和家庭统合综效的基础，能够使得整体获得一加一大于二的成效。
  4. 实践统合综效的人际关系和团队会扬弃敌对的态度（1+1=1/2），不以妥协为目标（1+1=3/2），也不仅仅止于合作（1+1=2），而是真正的创造式合作（1+1>2）。

习惯七：不断更新

  1. 这里不断更新是指：如何在四个生活基本面（身体、精神、智力、社会/情感）中，不断更新自己。
  2. 这个习惯提升了其他六个习惯的实施效率。
  3. 对组织而言，习惯七提升了愿景、更新及不断的改善，使组织不至呈现老化及疲态，并迈向新的成长之路。
  4. 对于家庭而言，习惯七通过固定的个人及家庭活动，使家庭能效升级，就像建立传统，使家庭日新月异，即是一例。
