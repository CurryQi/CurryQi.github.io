Ruby 正则表达式

正则表达式是一种特殊序列的字符，用于模式匹配和查询

语法

Ruby正则表达式形式上有两种

  • 一种介于斜杠之间
  • 另一种介于跟在 %r 后的任意分隔符之间

正则表达式修饰符

用于控制各方面的匹配

修饰符 描述
i 当匹配文本时忽略大小写
o 只执行一次 #{} 插值，正则表达式在第一次时就进行判断
x 忽略空格，允许在整个表达式中放入空白符和注释
m 匹配多行，把换行字符识别为正常字符
u,e,s,n 把正则表达式解释为 Unicode（UTF-8）、EUC、SJIS 或 ASCII。如果没有指定修饰符，则认为正则表达式使用的是源编码

正则表达式模式

  • 除了控制字符，(+ ? . * ^ $ ( ) [ ] { } | )，其他所有字符都匹配本身。
  • 对于控制字符，需在前面放置一个反斜杠来对其进行转义。
模式 说明
/a/ 精确匹配，匹配字符a
/?/ 转译字符匹配
. 匹配任意单个字符，如/a./匹配ab、ac等
/[ab]c/ 匹配ac和bc，[]之间代表范围，例如/[a-z]/，/[a-zA-Z0-9]/等
/^a-zA-Z0-9/ 匹配不在该范围内的字符串
/[\d]/ 代表任意数字
/[\w]/ 代表任意字母、或数字
/[\s]/ 代表空白字符，表示空格，TAB或换行
/[\D]/,/[\W]/,/[\S]/ 均为上述的否定情况
? 代表0个或1个字符
* 代表0个或多个字符
+ 代表1个或多个字符
/d{3}/ 匹配三个数字
/d{1, 10}/ 匹配1-10个数字
/d{3, }/ 匹配3个是数字以上
/([A-Z]\d){5}/ 匹配首位是大写字母，后面是4个数字的字符串

匹配方法

  1. match方法：匹配成功时返回MatchData类的实例，失败返回nil

    • pre_match方法，返回实例中匹配串前的串

    • post_match方法，返回实例中匹配串后的串

  2. =~操作符：匹配成功时返回位置索引（整数），失败返回nil

其他字符串常见方法

  1. String.scan(Patten)：尽可能多的去匹配，并将第一个匹配添加到数组中
  2. String.split(Patten)：基于Patten分割字符串，如果Patter==//为空，则将原串分割为字符。
  3. String.slice(Patten)：返回匹配的串，原串不变
  4. String.slice!(Patten)：返回匹配的串，并再原串中删除匹配的串（修改了原串）
